Here are the top most Git Commands
1.  **git add .** - This command is used to add all the changes in the current directory to the staging area.
2.  **git commit -m "commit message"** - This command is used to commit the changes in the staging area with a commit message.
3.  **git log** - This command is used to view the commit history of the repository.   
4.  **git branch** - This command is used to create a new branch in the repository.
5.  **git checkout** - This command is used to switch to a different branch in the repository.
6.  **git merge** - This command is used to merge two or more branches in the repository.
7.  **git remote add origin <repository URL>** - This command is used to link the local repository with a remote repository.
8.  **git push origin master** - This command is used to push the changes in the local repository to the remote repository.
9.  **git pull origin master** - This command is used to pull the changes from the remote repository to the local repository.
10. **git status** - This command is used to view the status of the repository, including any changes that have not been committed.
